# PUC – GitLab Scrum Analytics

Repositório desenvolvido como parte do Trabalho de Conclusão de Curso da Pós-Graduação em Engenharia de Dados da PUC.

## Descrição do Projeto
Este projeto tem como objetivo analisar a produtividade e o desempenho em processos ágeis de desenvolvimento de software a partir de dados extraídos do GitLab, utilizando Python para ingestão e enriquecimento dos dados e Power BI para análise e visualização.

O estudo de caso simula um ambiente corporativo fictício, aplicando práticas de Scrum e Software Analytics para suporte à tomada de decisão.

## Estrutura do Repositório
- `script/`  
  Contém o script Python responsável pela extração e enriquecimento dos dados via API do GitLab.

- `data/`  
  Contém o arquivo CSV gerado pelo processo de ingestão de dados, utilizado como fonte no Power BI.

- `powerbi/`  
  Arquivo do dashboard desenvolvido no Power BI, com indicadores de produtividade, complexidade e tarefas atrasadas.

- `docs/`  
  Imagens e diagramas utilizados como evidência visual do projeto e do fluxo de dados.

## Tecnologias Utilizadas
- Python
- GitLab API
- Power BI
- CSV como formato intermediário de dados

## Principais Análises
- Tarefas por colaborador
- Complexidade e peso das demandas
- Tarefas concluídas e atrasadas
- Indicadores de produtividade em sprints
